# Insurance-ML-Project
# 🏥 Insurance Cost Prediction using Machine Learning

##  Project Overview
This project predicts an individual's medical insurance charges based on demographic and lifestyle factors such as age, gender, BMI, number of children, smoking status, and region.

The project demonstrates the complete Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and model evaluation.


##  Dataset
The dataset contains the following features:

- Age
- Sex
- BMI
- Children
- Smoker
- Region
- Insurance Charges (Target Variable)

## 📈 Evaluation Metrics
